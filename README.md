# DynamicChartBuilder

DynamicChartBuilder is a web application that allows users to connect to their own database, retrieve data through functions, procedures, or views, and dynamically generate various types of charts (Line Chart, Bar Chart, Radar Chart). The application is designed to provide a user-friendly interface for data visualization.

## Features

- Connect to any database using user-provided connection details.
- Retrieve data using SQL functions, stored procedures, or views.
- Generate dynamic charts such as Line, Bar, and Radar charts based on the selected data.
- User-friendly interface with responsive design.

## Technologies Used

- **Frontend:** ASP.NET MVC, HTML, CSS, Bootstrap, jQuery
- **Charts:** Chart.js
- **Backend:** ASP.NET 8.0, Web API
